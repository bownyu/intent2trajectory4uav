### 1.悬停(Hover)
- **文本特征:** 位置不变、速度低。
- **生成逻辑:** 理想悬停是一个固定的点。为了增加真实感和样本多样性，引入三维空间的三维布朗运动或低频随机游走/一阶滤波噪声。
- **数学映射:**
	- 选取一个悬停中心点 $P_{center}$。
	$$P(t) = P_{center} + \mathcal{N}(0, \Sigma_{hover})$$
	其中 $\Sigma_{hover}$ 是一个很小的协方差矩阵，确保“速度低”（相邻点位移极小）且“位置基本不变”。
### 2.直线突防(Straight-line Penetration)
- **文本特征:** 距离减小、平均速度高、航向稳定。
- **生成逻辑:** 沿起点到目标的直线向量匀速或变速运动。
- **数学映射:**
    计算指向目标的单位方向向量 $\vec{u} = \frac{P_{target} - P_{start}}{||P_{target} - P_{start}||}$。
    在指定的高速区间 $[V_{min}, V_{max}]$ 内随机采样一个基础速度 $v$。
    $$P(t) = P_{start} + \int_{0}^{t} v(\tau) \vec{u} \, d\tau$$
    
    为了不那么刻板，$v(\tau)$ 可以是一个均值较高的高斯过程，但方向向量 $\vec{u}$ 严格保持不变（保证航向绝对稳定）。
### 3.非直线突防(Non-straight-line Penetration)
- **文本特征:** 距离减小、平均速度高、航向不稳定。
- **生成逻辑:** 这是最复杂的。需要保证总体趋势是向目标靠近（距离减小），但局部路径是曲折的（如蛇形、Z字形、螺旋形）。
- **数学映射 (正弦机动法):**
    设定基础突防方向 $\vec{u}$（指向目标）和与其垂直的横向向量 $\vec{w}$。
    设置前向高速度 $v_{fwd}$，以及横向正弦震荡的振幅 $A$ 和角频率 $\omega$。
$$P(t) = P_{start} + (v_{fwd} \cdot t)\vec{u} + A \sin(\omega t)\vec{w}$$
	通过随机改变振幅 $A$ 和频率 $\omega$（或者加入多个正弦波叠加），可以生成各种航向不断变化，但最终“距离减小”的轨迹。
### 4.盘旋(Loiter)
- **文本特征:** 航向不稳定、长窗口平均速度大、高度变化范围小。
- **生成逻辑:** 在二维水平面上进行圆周或椭圆运动，Z轴保持在一个小范围内波动。
- **数学映射:**
    随机生成盘旋中心 $P_{center} = (x_c, y_c, z_c)$，盘旋半径 $R$，和角速度 $\omega$（对应较高的线速度 $v = \omega R$）。
    $$x(t) = x_c + R \cos(\omega t + \phi_0)$$
    $$y(t) = y_c + R \sin(\omega t + \phi_0)$$
    $$z(t) = z_c + \mathcal{N}(0, \sigma_z^2)$$
    连续的圆周运动自然满足了“航向不断变化”和“高度变化小”的约束。
### 5. 撤退 (Retreat)
- **文本特征:** 远离、距离增大占比大90%。
- **生成逻辑:** 总体向量背离目标点，允许有局部的机动或规避，但宏观上距离必须拉开。
- **数学映射:**
    计算背离目标的单位方向向量 $\vec{u}_{away} = \frac{P_{start} - P_{target}}{||P_{start} - P_{target}||}$。
    为了满足“占比90%”这个统计特征，我们可以使用一种带有前向偏置的随机游走模型。
	$$P(t) = P(t-\Delta t) + [v_{base} \vec{u}_{away} + \vec{noise}(t)] \Delta t$$
    调整 $v_{base}$（基础撤退速度）和 $\vec{noise}(t)$（随机机动向量）的比例。只要 $v_{base}$ 足够大，在绝大多数时间步 $t$ 内，相对于目标点的距离的导数 $\frac{d}{dt}||P(t) - P_{target}||$ 都会大于 0。如果生成的轨迹经计算发现“距离增大比例”不足 90%，则丢弃该样本重新生成。

### 确保轨迹生成质量的关键控制点

1. **连续性与平滑度 (Smoothness):** 纯数学公式（特别是加入随机噪声时）容易产生“折线”。为了更符合物理现实（即使不跑 PX4），建议在生成离散点后，使用 **B样条曲线 (B-Spline)** 或 **贝塞尔曲线 (Bézier Curve)** 进行一次平滑拟合，保证轨迹在数学上至少是一阶或二阶可导的。
2. 对 pos 序列计算 vel/acc/yaw_rate，若超过你们设定的消费级机体边界（Vmax/amax/偏航角速度），就**自动缩放/重采样/平滑**再送入仿真（否则会频繁触发限幅，导致类特征被飞控“抹平”）
3. **边界条件约束 (Boundary Limits):** 无论怎么随机生成，都需要加一层硬约束（Clip）：
    - 最大速度约束：$\frac{||P_i - P_{i-1}||}{\Delta t} \leq V_{max\_drone}$。
    - 飞行空域约束：$x, y, z$ 必须在指定的仿真区域内，且 $z > 0$（不能钻入地下）。
